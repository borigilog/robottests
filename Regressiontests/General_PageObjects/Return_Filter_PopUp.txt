    ...    - @{select_reason_names} = list of reason names to be selected
    ...    == Example ==
    ...    @{reason_names}= Create List Materialfehler Materialdefekt
    ...    | = Keyword = | = locator = | = select_reason_names = | = Result = |
    ...    | Set ReturnReasons Selection In ReturnFilter PopUp | /$flow_main/$flow_content | @{reason_names} | select reasons |
    [Tags]    General, Helper, ReturnFilter_PopUp
    ${grid_id}=    Get FilterReturnReason GridID    ${locator}
    @{values}=    Get CheckboxIds With Status For Table Column    ${grid_id}    chk_return_reason    txt_return_reason
    : FOR    ${checkbox}    IN    @{values}
    \    ${chkbox_id}=    Get From Dictionary    ${checkbox}    id
    \    ${text}=    Get From Dictionary    ${checkbox}    text
    \    ${select} =    Run Keyword And Return Status    List Should Contain Value    ${select_reason_names}    ${text}
    \    Set Selection of CheckBox    ${chkbox_id}    ${select}

Get ReturnReasons Selection In ReturnFilter PopUp
    [Arguments]    ${locator}
    [Documentation]    MM 12.04.2018
    ...
    ...    == Parameters ==
    ...    - ${locator} = the general path of all filter elements , e.g.: /$flow_main/$flow_content/filters \ = /$flow_main/$flow_content
    [Tags]    General, Helper, ReturnFilter_PopUp
    ${grid_id}=    Get FilterReturnReason GridID    ${locator}
    @{values}=    Get CheckboxIds With Status For Table Column    ${grid_id}    chk_return_reason    txt_return_reason
    ${selection}=    Create List    @{EMPTY}
    : FOR    ${checkbox}    IN    @{values}
    \    ${selected}=    Get From Dictionary    ${checkbox}    selected
    \    Continue For Loop If    '${selected}'=='${FALSE}'
    \    ${text}=    Get From Dictionary    ${checkbox}    text
    \    Append To List    ${selection}    ${text}
    [Return]    ${selection}

    ...    == Example ==
    ...    @{reason_names}= Create List Materialfehler Materialdefekt
    ...    | = Keyword = | = locator = | = select_reason_names = | = Result = |
